Decimal Pipe

{{ { minIntegerDigits }.{ minFractionDigits }-{ maxFractionDigits } }}

  1. minIntegerDigits 是最小的整数位数，默认是1。
  2. minFractionDigits 表示最小的小数位数，默认是0。
  3. maxFractionDigits 表示最大的小数位数，默认是3。
